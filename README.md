# Glamora - AI-Powered Fashion E-commerce App

## About

Glamora is a modern, feature-rich fashion e-commerce mobile application built with Flutter. The app combines traditional online shopping with cutting-edge AI technology to provide users with personalized fashion recommendations, virtual try-on experiences, and intelligent styling assistance. Glamora offers a seamless shopping experience for fashion enthusiasts, featuring an extensive catalog of clothing items including jeans, t-shirts, sweatshirts, and accessories for both men and women.

## Getting Started

### Prerequisites
- Flutter SDK (latest stable version)
- Dart SDK
- Android Studio / VS Code
- Android/iOS development environment setup

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/glamora.git
cd glamora
```

2. Install dependencies
```bash
flutter pub get
```

3. Configure environment variables
- Set up Google AI API keys
- Configure PayPal and Stripe credentials
- Add necessary API endpoints

4. Run the application
```bash
flutter run
```
